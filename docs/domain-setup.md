## Passo 1 — Comprar (5 min)

### Opção A — pelo site (mais fácil)

1. Abra: https://vercel.com/dashboard/domains  
2. Clique em **Buy** / **Add Domain**  
3. Digite: `pedromauri.com`  
4. Confirme o preço (~$11.25), auto-renew e dados de contato  
5. Pague  
6. **Não** compre hospedagem Hostinger

### Opção B — pelo terminal (no PowerShell)

```powershell
cd c:\Users\pedro\pedro-portfolio
npx vercel domains buy pedromauri.com
```

Responda às perguntas (preço, renovação, contato).

## Passo 2 — Ligar ao portfolio (2 min)

1. Abra: https://vercel.com/pedro-mauri-6956s-projects/pedro-portfolio/settings/domains  
2. Adicione: `pedromauri.com`  
3. Adicione: `www.pedromauri.com`  
4. Se o domínio foi comprado **na própria Vercel**, o DNS costuma configurar **sozinho**  
5. Defina um como principal (ex.: `pedromauri.com`) e redirecione `www` para ele
