# Yethos case study assets

Export these frames from Figma and place them in this folder:

| File | Description |
|------|-------------|
| `hifi-community.png` | Final high-fidelity community page |
| `wireframe-homepage.png` | Early wireframe exploration |
| `sitemap-ia.png` | Information architecture / site map |
| `user-flow.png` | Primary user flow diagram |
| `project-workflow.png` | Full project workflow |

Recommended: PNG or WebP at 2x scale (~1440px wide for hi-fi).

Once added, commit and push — the site will display them automatically.
